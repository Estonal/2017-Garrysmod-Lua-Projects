Garry's mod 게임에서 Lua 프로그래밍으로 개발한 lua script 파일들

2016~2017(중학교)년에 멀티플레이어 나만의 게임모드를 개발해보고 직접 서버를 운영해보고 싶었음.
그러던 중 한 스팀 유저분께서 감사하게도 Gmod Lua에 대해 나에게 가르쳐주겠다고 하셨음.
사실상 독학이지만, 그 스승님은 나에게 매번 '일'이라는 것을 주셔서 Lua 실력을 키울 수 있도록 도와주셨음.

각 프로젝트에 대한 설명

sniper - 라운드FPS gamemode : 레드팀, 블루팀이 나뉘어 제한시간 안에 상대 팀과 총으로 겨루는 게임모드. Counter Strike : Source의 게임 규칙과 거의 유사하다. 라운드 시스템이 적용되어 있다.

warnscript - 경고(warning)시스템 기능을 서버에 제공하는 스크립트. 채팅으로 '!경고'를 입력하면 서버 내 모든 유저들의 경고 누적 현황을 열람할 수 있음.
+ 관리자 권한을 소유한 경우, 경고 메뉴의 플레이어를 우클릭하면 경고 주기, 경고 초기화, 킥, 밴, 고유번호 복사(SteamID), 스팀 프로필 보기 등의 기능을 이용 가능함.
경고 누적에 따라 차등 벌이 주어짐. 벌의 강도는 구두경고부터 킥, 시간제 밴, 영구 밴 등 다양함.
해당 스크립트를 서버에 적용할 경우, 서버의 규칙을 어기거나 예의를 지키지 않는 무례한 사람들을 처리할 수 있을 것으로 기대됨.


해당 게임모드, 스크립트를 적용하여 실제 서버를 운영해본 결과, 당시 상당한 인기를 끌었고, 이것에 대해 뿌듯함을 느꼈음. 유저들의 의견이 게임모드나 스크립트 개발에 도움이 되는 것이 재미있었고, 원하는 대로 구현하거나 새롭게 코드를 짜 주면서 코드 구현력도 기를 수 있었음.

게시한 소스코드 이외에도 욕 차단 시스템, 칭호, 파티, 길드 시스템 등 여러 프로젝트를 구현했지만, 현재는 모두 잃어버려 안타까움. 만약 다시 찾게 된다면 commit 할 생각이 있음.
